# 지역별 인구구조 대시보드

행정안전부 '주민등록 연령별 인구현황(월간)' 데이터를 이용해 지역별 연령 구조를 비교하고, 전국에서 인구구조가 가장 비슷한 지역을 찾아주는 Streamlit 앱입니다.

## 주요 기능

### 1. 지역별 인구구조 비교
- 지역명을 검색(입력)하거나 목록에서 선택(select)해 여러 지역을 동시에 비교
- 성별(전체/남/여)별로 0~100세(100세는 100세 이상 합산) 인구수를 Plotly 꺾은선 그래프로 표시
- 데이터 테이블로 원본 수치 확인 가능

### 2. 전국 유사 인구구조 찾기
- 기준 지역(시/도, 시/군/구, 읍/면/동 모두 포함)을 검색·선택
- 전국 모든 지역과 연령대별 인구 비율(구성비)을 비교해 가장 비슷한 지역 Top N(3~10개, 기본 5개) 산출
- 유사도 기준 2가지 중 선택 가능
  - 유클리드 거리(기본, 값이 작을수록 유사)
  - 코사인 유사도(값이 1에 가까울수록 유사)
- 결과를 순위 테이블, 유사도 막대그래프, 기준 지역과 Top N의 연령 비율(%) 오버레이 꺾은선 그래프로 시각화

## 데이터

| 항목 | 내용 |
|---|---|
| 출처 | 행정안전부 주민등록 연령별 인구현황(월간) |
| 파일명 | `202606_202606_연령별인구현황_월간.csv` |
| 인코딩 | cp949 (euc-kr / utf-8도 자동 시도) |
| 구조 | 행: 행정구역(시/도 · 시/군/구 · 읍/면/동), 열: `{연월}_{계|남|여}_{연령}세` 형식의 인구수 |

앱이 CSV 컬럼명을 정규식으로 파싱하기 때문에, 같은 형식(연월_성별_연령세)이면 다른 월/지역 데이터로 교체해도 그대로 동작합니다.

## 프로젝트 구조

```
.
├── main.py                                # 메인 대시보드 (기능 1, 2 포함)
├── requirements.txt
├── 202606_202606_연령별인구현황_월간.csv     # main.py와 같은 폴더(리포 루트)에 위치
└── pages/
    └── 00_쌍둥이동네찾기.py                  # 기능 2(유사 지역 찾기)를 별도 페이지로 분리한 버전
```

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run main.py
```

## Streamlit Cloud 배포

1. `main.py`, `requirements.txt`, CSV 데이터 파일을 **모두 같은 폴더(리포 루트)** 에 커밋합니다.
2. [share.streamlit.io](https://share.streamlit.io) 에서 New app → 리포 선택 → Main file path에 `main.py` 지정 후 Deploy.
3. 코드나 requirements.txt를 나중에 수정했는데 반영이 안 되면 앱 관리 화면에서 **Reboot app**을 눌러주세요.

## 트러블슈팅

**`ModuleNotFoundError` (plotly 등)**
`requirements.txt`가 리포 루트에 정확한 파일명으로 있는지 확인하고, 앱을 Reboot 하세요.

**"데이터 CSV 파일을 찾을 수 없습니다"**
- CSV가 실제로 커밋됐는지 확인
- 한글 파일명은 OS/깃 클라이언트에 따라 유니코드 정규화(NFC/NFD) 방식이 달라 화면상 이름이 같아도 못 찾는 경우가 있음 → 앱이 정규화 비교로 자동 보정
- `pages/` 하위 스크립트는 자기 폴더뿐 아니라 상위 폴더(리포 루트)까지 자동 검색하도록 처리되어 있어 멀티페이지 구조에서도 동작
